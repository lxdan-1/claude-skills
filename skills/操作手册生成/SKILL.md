# 操作手册生成

## 使用目标

接收大屏、页面或系统截图，自动识别图中的指标名称、图表类型、数据展示方式，根据实际内容补充或生成结构化的操作手册文档。

## 输入前提

优先接收以下输入：

- 页面/大屏截图（PNG、JPG 等图片格式）
- 页面名称或功能描述
- 现有操作手册文档（如有）
- 补充说明或特殊要求

## 适用场景

- 需要根据实际页面截图生成操作手册
- 需要识别页面顶部的具体指标名称（如：总病历数、异常金额、拒付率等）
- 需要识别页面中的图表类型（柱状图、圆形饼图、折线图、地图、关系图等）
- 需要补充操作流程的详细步骤和数据展示说明
- 需要确保操作手册与实际页面功能一致

## 执行步骤

1. **图片分析**：仔细查看截图，识别页面的整体布局和各个区域
2. **指标提取**：提取页面顶部或关键位置显示的具体指标名称和数值
3. **图表识别**：识别页面中各个图表的类型（柱状图、饼图、折线图、地图、表格等）
4. **内容梳理**：整理页面的主要功能模块和交互方式
5. **文档补充**：根据识别结果补充或生成操作手册的详细内容
6. **格式规范**：按照操作手册的标准模板格式输出

## 页面分析清单

分析截图时，优先覆盖这些维度：

- **页面标题**：页面的名称和功能定位
- **顶部指标**：页面顶部显示的关键数据指标（从左到右）
- **左侧区域**：左侧的数据分析模块和图表类型
- **中心区域**：中心的主要展示内容（通常是地图或大图表）
- **右侧区域**：右侧的补充分析模块和图表类型
- **底部区域**：底部的补充信息或操作区域
- **交互元素**：按钮、筛选器、页签、下拉菜单等交互元素
- **数据展示**：具体的数据字段、数值、时间范围等

## 操作手册标准模板

```markdown
### X.X.X **页面名称**

***\*页面简介\****

简要说明页面的功能和用途。

***\*操作流程\****

 \1. 页面入口

从【菜单名称】菜单进入页面名称

 \2. 数据展示

**顶部关键指标**（从左到右）：
- 指标1名称：指标1说明
- 指标2名称：指标2说明
- 指标3名称：指标3说明

**左侧数据分析区**：
- 模块1名称（图表类型）：模块1说明
- 模块2名称（图表类型）：模块2说明

**中心展示区**：
- 地图/主图表说明

**右侧数据分析区**：
- 模块3名称（图表类型）：模块3说明
- 模块4名称（图表类型）：模块4说明

 \3. 交互操作

- 交互1说明
- 交互2说明
- 交互3说明
```

## 指标识别规则

识别顶部指标时，注意：

- 指标通常以卡片或数字形式展示
- 指标名称可能包含单位（如：份、个、万元、%等）
- 指标可能有图标或颜色标识
- 指标顺序从左到右依次记录
- 如果指标有变化趋势（↑↓），也要记录

## 图表类型识别

常见图表类型及其特征：

- **柱状图**：竖向或横向的矩形条形
- **圆形饼图**：圆形分割图，显示占比
- **折线图**：连接的点形成的曲线
- **地图**：地理位置分布展示
- **表格**：行列结构的数据展示
- **关系图**：节点和连线的网络结构
- **3D立方体**：三维展示对比
- **气泡图**：不同大小的圆形气泡
- **卡片**：矩形信息卡片
- **环形图**：圆形环状的占比展示

## 输出要求

默认输出以下结构：

1. **页面简介**：一句话说明页面功能
2. **操作流程**：
   - 页面入口
   - 数据展示（按区域分类）
   - 交互操作
3. **具体指标和图表说明**：
   - 每个指标的具体名称
   - 每个图表的类型和用途
   - 数据展示的逻辑

## 质量检查

- 是否准确识别了所有顶部指标的具体名称
- 是否正确识别了所有图表的类型
- 是否清楚说明了各个模块的功能和用途
- 是否按照标准模板格式输出
- 是否与实际页面功能一致
- 是否补充了具体的交互操作说明
- 是否避免了模糊表达（如"展示数据"应改为"展示患者总数、费用总额等"）

## 使用示例

### 输入
用户上传一张大屏截图，并说明："这是运行监测大屏，请根据图片内容补充操作手册"

### 处理流程
1. 分析截图，识别顶部6个指标：医保区划数、定点医疗机构数、风险病例总数、总住院费用、拒付率、拒付金额
2. 识别左侧3个图表：监管异常机构TOP5（柱状图）、监管问题占比（圆形饼图）、监管问题变化趋势（折线图）
3. 识别中心地图展示
4. 识别右侧3个图表：DIP病种监管异常TOP5（横向柱状图）、异地就医问题占比（圆形饼图）、异地就医监管问题趋势（折线图）
5. 输出结构化的操作手册内容

### 输出
按照标准模板输出详细的操作手册，包含所有识别的指标名称、图表类型和交互说明。
