# 操作手册生成示例

## 示例1：根据运行监测大屏截图生成操作手册

### 输入

用户上传运行监测大屏截图，并说明："这是九江市大数据反欺诈运行监测大屏，请根据图片内容补充操作手册"

### 分析过程

1. **页面标题识别**：九江市大数据反欺诈运行监测
2. **顶部指标识别**（从左到右）：
   - 医保区划数：11
   - 定点医疗机构数：53个
   - 风险病例总数(份)：870,442份
   - 总住院费用：85.55亿元
   - 拒付率：322,929份
   - 拒付金额：57,613.47万元

3. **左侧区域识别**：
   - 监管异常机构TOP5（柱状图）
   - 监管问题占比（圆形饼图）：显示60%、20%、20%等占比
   - 监管问题趋势（折线图）

4. **中心区域识别**：
   - 地图展示各地区的监管分布
   - 鼠标悬停显示地区数据

5. **右侧区域识别**：
   - 监管问题DIP病种TOP5（横向柱状图）
   - 异地就医问题占比（圆形饼图）：显示60%、20%、20%等占比
   - 异地就医监管问题趋势（折线图）

### 输出

```markdown
### 1.1.3 **九江市大数据反欺诈运行监测**

***\*页面简介\****

实时展示医保系统运行状态，包括病种监管、疑似违规、拒付等多维度数据统计，支持按地区和时间范围查看监测数据。

***\*操作流程\****

 \1. 页面入口

从【综合决策大厅】菜单进入运行监测

 \2. 数据展示

**顶部关键指标**（从左到右）：
- 医保区划数：医保覆盖的行政区划总数
- 定点医疗机构数：纳入医保的定点医疗机构总数
- 风险病例总数(份)：识别出的风险病例总数
- 总住院费用：统计周期内的总住院费用
- 拒付率：拒付病例占比
- 拒付金额：拒付的总金额

**左侧数据分析区**：
- 监管异常机构TOP5（柱状图）：展示异常金额最高的5家机构及其具体异常金额
- 监管问题占比（圆形饼图）：展示各类监管问题（如过度医疗、分解住院等）的占比分布
- 监管问题变化趋势（折线图）：展示最近6个月监管问题的变化趋势

**中心地图展示**：
- 地图展示各地区的监测数据分布
- 鼠标悬停显示该地区的具体数据
- 点击地区可查看该地区的详细监测信息

**右侧数据分析区**：
- DIP病种监管异常TOP5（横向柱状图）：展示异常金额最高的5个DIP病种及其异常金额
- 异地就医问题占比（圆形饼图）：展示异地就医的各类问题占比
- 异地就医监管问题趋势（折线图）：展示异地就医问题的变化趋势

 \3. 交互操作

- 点击地图上的地区可查看该地区的详细数据
- 点击图表中的数据可进行钻取查看
- 支持时间范围选择查看不同时期的数据
```

## 示例3：根据AI模式对比大屏截图生成操作手册

### 输入

用户上传AI模式与传统模式监测大屏截图，并说明："这是AI模式与传统模式对比大屏，请根据图片内容补充操作手册"

### 分析过程

1. **页面标题识别**：AI模式与传统模式监测
2. **左侧区域识别**：
   - 审核能力对比（圆形饼图）：AI模式85.30%、传统模式14.70%等
   - 审核结果对比（圆形饼图）
   - 审核结果对比（圆形饼图）

3. **中心区域识别**：
   - AI模式与传统模式对比（3D立方体展示）
   - 支持旋转和交互查看

4. **右侧区域识别**：
   - 审核效率对比（柱状图）
   - 审核异常检出TOP5对比（柱状图）
   - 选择类型检出TOP5对比（柱状图）

### 输出

```markdown
### 1.1.12 **AI模式与传统模式监测**

***\*页面简介\****

对比展示AI审核模式和传统审核模式的监测数据，包括审核能力对比、审核结果对比、成效对比等。

***\*操作流程\****

 \1. 页面入口

从【综合决策大厅】菜单进入AI模式与传统模式监测

 \2. 数据展示

**左侧对比分析区**：
- 审核能力对比（圆形饼图）：展示AI模式和传统模式的审核能力占比对比
- 审核结果对比（圆形饼图）：展示两种模式的审核结果对比
- 审核结果对比（圆形饼图）：展示两种模式的审核结果占比

**中心3D展示区**：
- AI模式与传统模式对比（3D立方体）：直观展示两种模式的多维度对比
- 支持旋转和交互查看

**右侧对比分析区**：
- 审核效率对比（柱状图）：展示两种模式的审核效率对比
- 审核异常检出TOP5对比（柱状图）：展示两种模式检出的异常TOP5对比
- 选择类型检出TOP5对比（柱状图）：展示两种模式在不同选择类型上的检出对比

 \3. 交互操作

- 点击图表可查看详细的对比数据
- 支持切换不同的对比维度
- 鼠标悬停显示具体数值
```

## 使用提示

- 每次接收新的截图时，优先分析页面的整体布局和各个区域
- 准确识别顶部指标的具体名称，避免模糊表达
- 正确识别图表类型，这是补充操作手册的关键
- 按照标准模板格式输出，确保与现有操作手册风格一致
- 如果图片中有文字标签，优先使用图片中的原始文字
- 如果某些指标或图表不清晰，可标注为"待确认"
